## Features

<ul>
    <li>Personalized diet and workout plan based on user's weight, height, age, gender, and number of workout days per week</li>
    <li>Streak tracking to keep users motivated</li>
    <li>Notifications to remind users about workouts or meals</li>
    <li>AI-powered scheduling using agents (Gemini) with a highly accurate dataset</li>
    <li>Workout schedule includes timers and can be followed directly on the website</li>
    <li>Notifications are sent to keep the user highly motivated</li>
</ul>
